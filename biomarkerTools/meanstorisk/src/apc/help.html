This application accepts data formatted in columns organized into Count and Population for a given period. An example data matrix is shown on the right.

Demonstration:

  1. Enter input values as follows
    Name: Female Pancreas Cancer
    Description: SEER13, Females only, Whites only, age 33-85
    Start Year: 1993
    Start Age: 33
    Interval: 4 years
  2. Click on the Browse button load in a cvs file that contains data formatted like the example data matrix (shown on the right). Content of the csv file would be copied into the input table under the Input tab.
      Or
    Copy and paste the example data matrix (shown on the right) to the input table under the Input tab.
  3. Click on the Calculate button below the input table and wait for the calculation to complete.
  4. Click through the tabs on the right to observe results
