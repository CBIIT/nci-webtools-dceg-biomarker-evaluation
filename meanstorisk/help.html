Instructions for input
  • Input from raw Excel data:
    The .csv file must be in the format where the 2nd column is the outcome, 0 or 1, and the first row includes the column names.
    1. Click Browse and open your .csv file.
    2. Enter your specificity and prevalence values as decimals separated by commas.
    3. Click Calculate.


  • Input from normally distributed biomarker data:
    The data must be normally distributed. If the data is not normally distributed, the calculations may be inaccurate.
    1. Enter the means, standard errors, and N values for both cases and controls.
    2. Enter your specificity and prevalence values as decimals separated by commas.
    3. Click Calculate.

Where do I go for technical support? Please send an e-mail to our technical support team.

What browsers does the web tool support? The web tool has been tested to work with Internet Explorer 10, Firefox and Chrome. The file upload feature currently does not work with Internet Explorer 9 and below.
